4.2.2 正規分布を仮定したときの平均の標本分布(p.109-111)

  • 「平均の標本分布」はRでの演習はありません。

  • 母集団分布が正規分布である場合には，その標本分布自体が正規分布となることが知られています。

  • 図4-6について，Rでのシミュレーションを通して確認してみましょう。ただし，母集団分布を標準正規分布と仮定します。)

    Rで正規分布に従う乱数を生成するには，rnorm()関数を利用します。rnorm(発生させる乱数の数，母集団の平均値，標準偏差)を与えることで，正規分布に従う乱数を生成することができます。


  • シミュレーションの手順

    1. 正規分布に従う乱数を生成し，その平均\(\bar{x}\)を計算します。
    2. 1.の手順を1000回繰り返し，1000個の標本平均\(\bar{x}\)を求めます。
    3. 求めた標本平均\(\bar{x}\)の平均が，母集団の平均値に一致するか，母集団の標準偏差の\(1 / \sqrt{N}\)倍に案っているか確認します。

  • \(N = 1\)のとき

    • 標準正規分布に従う乱数を1個取り出す作業を1000回繰り返し，std.norm.1というオブジェクトに代入します。
set.seed(1234)
std.norm.1 <- NA
for(i in 1:1000){
  std.norm.1[i] <- rnorm(1, mean = 0, sd = 1)
}
  • \(N = 1\)の時は平均を求める必要がない(1で割ることになる)ので，std.norm,1がそのまま標本平均となります。

  • 標本平均\(\bar{x}\)の平均と標準偏差を求めます。
mean(std.norm.1)
## [1] -0.0265972
sd(std.norm.1)
## [1] 0.9973377

  • 標本平均\(\bar{x}\)の平均は0，標準偏差は\(\frac{1}{\sqrt{1}} = 1\)に近い値となっていることがわかります。

  • 得られた1000個の標本平均をヒストグラムで示します。

hist(std.norm.1, freq = FALSE, xlim = c(-4,4))

  • 標本平均\(\bar{x}\)のヒストグラムは，左右対称の釣り鐘型になっています。


  • \(N = 3\)のとき

    • 乱数を3個取り出す作業を1000回繰り返し，std.norm.3というオブジェクトに代入します。
set.seed(1234)
std.norm.3 <- matrix(NA, ncol = 3, nrow = 1000)
for(i in 1:1000){
  std.norm.3[i, ] <- rnorm(3, mean = 0, sd = 1)
}
  • コンソールにstd.norm.3と直接入力すると，乱数が確認できます。

  • 次に，3個の乱数の標本平均を求め，std.norm.3_meanというオブジェクトに代入します。データとしては，行方向に平均をとることと同じなので，rowSums()関数を用います。

行列形式のデータで平均を求めるときには，行平均はrowMeans()関数，列平均はcolMeans()関数を使います。

std.norm.3_mean <- rowMeans(std.norm.3)
  • 最後に標本平均\(\bar{x}\)の平均と標準偏差を求めます。
mean(std.norm.3_mean)
## [1] 0.005620729
sd(std.norm.3_mean)
## [1] 0.5702766

  • 標本平均\(\bar{x}\)の平均は0付近，標準偏差は\(1 / \sqrt{3}\)の0.577に近い値となっています。

  • 得られた1000個の標本平均をヒストグラムで示します。

hist(std.norm.3_mean, freq = FALSE, xlim = c(-4,4))

  • \(N = 1\)の時と比較して，母集団平均\(\mu\)に集中した形になっていることがわかります。

  • \(N = 10\)のとき

    • 乱数を10個取り出す作業を1000回繰り返し，std.norm.10というオブジェクトに代入します。
set.seed(1234)
std.norm.10 <- matrix(NA, ncol = 10, nrow = 1000)
for(i in 1:1000){
  std.norm.10[i, ] <- rnorm(10, mean = 0, sd = 1)
}
  • 10個の乱数の標本平均を求め，std.norm.10_meanというオブジェクトに代入します。
std.norm.10_mean <- rowMeans(std.norm.10)
  • 最後に標本平均\(\bar{x}\)の平均と標準偏差を求めます。
mean(std.norm.10_mean)
## [1] 0.006115893
sd(std.norm.10_mean)
## [1] 0.3030795
sqrt(mean((std.norm.10_mean - mean(std.norm.10_mean))^2))
## [1] 0.3029279

  • 標本平均\(\bar{x}\)の平均は0付近，標準偏差は\(1 / \sqrt{10}\)の0.316に近い値となっています。

  • 得られた1000個の標本平均をヒストグラムで示します。

hist(std.norm.10_mean, freq = FALSE, xlim = c(-4,4))



  • 平均50，標準偏差10の正規分布の正規母集団からサンプルをとったとき，その平均\(\bar{x}\)が母集団平均50から1以上離れた値をとる確率(N = 25のとき)

pnorm()関数はある値までの確率を求めます。ある値以上の確率を求めるには，すべての確率を足し合わせると1になるという性質を利用します。

pnorm(49, mean = 50, sd = 10 / sqrt(25)) + (1 - pnorm(51, mean = 50, sd = 10 / sqrt(25)))
## [1] 0.6170751
  • 教科書の変形(\(Prob(z > .5) \times 2\))を使って以下のコードでも同じ結果となります。
(1 - pnorm(0.5, mean = 0, sd = 1)) *2
## [1] 0.6170751

補足(クリックすると説明が表示されます) - これまでpnorm()関数で上側確率を求めるには，1から引く方法のみを紹介してきました。
- pnorm()関数には，lower.tailという引数があります。この引数はデフォルトの設定(何も指定しないと，デフフォルトの設定になります)で，下側確率を求めるように指定されています。
- 上側確率を求めたいときには，lower.tail = FALSEと指定してください。

-以下の2つのコードは同じ結果が得られます。

1 - pnorm(1.96)
## [1] 0.0249979
pnorm(1.96, lower.tail = FALSE)
## [1] 0.0249979
